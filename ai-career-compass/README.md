# 🚀 AI Career Roadmap Generator

## 📌 Overview

AI Career Roadmap Generator is an AI-powered web application that helps users plan their career growth through personalized learning roadmaps. By analyzing a user's current role, target role, existing skills, and weekly learning commitment, the system generates a customized 60-day career development plan.

Powered by Google Gemini AI, the application provides skill gap analysis, recommended learning resources, project ideas, and a structured roadmap to help users achieve their career goals efficiently.

---

## ✨ Features

### 🎯 Personalized Career Roadmap

Generate a customized 60-day learning plan based on your career objectives.

### 📊 Skill Gap Analysis

Identify the skills you already possess and the skills required for your target role.

### 📚 Course Recommendations

Receive curated course suggestions to accelerate learning.

### 🛠️ Capstone Project Suggestions

Get real-world project ideas that strengthen your portfolio and practical knowledge.

### 📈 Learning Progress Planning

Adjust your weekly learning commitment and receive an optimized roadmap.

### 🤖 AI-Powered Guidance

Utilizes Google Gemini AI to provide intelligent career recommendations.

### 🎨 Modern User Interface

Responsive and interactive dashboard built using React and Tailwind CSS.

---

## 🏗️ System Workflow

1. Enter Current Role
2. Select Target Career Role
3. Provide Existing Skills
4. Choose Weekly Learning Hours
5. Generate AI-Powered Roadmap
6. View Skill Gap Analysis
7. Follow the 60-Day Learning Timeline
8. Complete Recommended Projects and Courses
